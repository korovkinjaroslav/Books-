Разработать настольное приложение на Python, реализующее рекомендательную систему книг.
Приложение должно предоставлять пользователю удобный графический интерфейс для просмотра, поиска,
оценки и получения персонализированных рекомендаций книг на основе данных из локальной базы данных.

1.Архитектура приложения
    Главное окно
        Кнопка "Мои рекомендации"
        Кнопка "Поиск по параметрам"
        Кнопка "Добавить книгу"
        Кнопка "Поиск книги"
        Кнопка "Прочитанные"
    Окно детального просмотра книги.
        Отображается автор книги, описание, жанр и обложка.
        Кнопка "Поменять обложку"
            Открывается диалоговое окно для выбора нового файла обложки
        Кнопка "Оценить книгу"
    Окно с отображением рекомендаций.
        Для каждой книги отображается автор, описание, жанр и обложка.
        Кнопка "Экспорт в csv"
    Окно отображения всех прочитанных книг
        Для каждой книги отображается автор, описание, жанр и обложка.

2.База данных
    Таблица books
        id
        isbn
        title
        autor
        category
        year
        description
        cover
        average_rating
    Таблица ratings
        id
        rating
    Таблица year_preferences
        year
        rating
    Таблица category_preferences
        category
        rating
    Таблица autor_preferences
        autor
        rating

3.Рекомендательная система
    Каждая книга оценивается пользователем от 1 до 10.

    Для каждого жанра, автора и периода времени длиной в 50 лет высчитывается средняя оценка книг,
    соответствующих им. По умолччанию средняя оценка равна 7.

    Рекомендательная система также учитывает среднюю оценку книги на сайте Goodreads

    При поиске книг по параметрам, пользователь получает книги, соответствующие заданным параметрам
    и его предпочтениям.

4.Библиотеки
    PyQt6 - GUI
    sqlite3 - работа с БД
    csv - для работы с csv файлами
